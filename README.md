# 주제형 한바퀴 보드게임 (교사/학생 분리)

## 페이지
- `index.html` : 진입(교사용/학생용 선택)
- `teacher.html` : 교사용(주제→AI 문제 생성→문제 파일 저장)
- `student.html` : 학생용(문제 파일 불러오기→게임 플레이, API 키 없음)

## 운영 권장
1) 교사: teacher.html에서 주제 입력 → 문제 적용(덱 생성) → 문제 파일 저장  
2) 학생: student.html에서 문제 파일 불러오기 → 2인 게임 진행

## 덱 소진형
문제 수(덱 문항 수)를 크게 만들면, 게임 중 문제칸에 도착할 때마다
덱에서 다음 문제가 순서대로 출제됩니다. 덱이 끝나면 안내가 뜹니다.
