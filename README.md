# 주제형 한바퀴 보드게임 (교사/학생 분리)

## 페이지
- `index.html` : 진입(교사용/학생용 선택)
- `teacher.html` : 교사용(주제→AI 문제 생성→문제 파일 저장)
- `student.html` : 학생용(문제 파일 불러오기→게임 플레이, API 키 없음)

## 운영 권장
1) 교사: teacher.html에서 주제 입력 → 문제 적용(덱 생성) → 문제 파일 저장  
2) 학생: student.html에서 문제 파일 불러오기 → 2인 게임 진행

## 덱 소진형
문제 수(덱 문항 수)를 크게 만들면, 게임 중 문제칸에 도착할 때마다
덱에서 다음 문제가 순서대로 출제됩니다. 덱이 끝나면 안내가 뜹니다.

## (v10) 무료 자동 결과 수집(권장): Google Sheets + Apps Script

학생이 파일을 저장/제출하지 않아도, 게임 중 정오답 기록이 자동으로 구글시트에 쌓이도록 할 수 있습니다.
Firebase 없이도 무료로 운영 가능합니다.

### 1) Google Apps Script 배포
1. Google Drive에서 새 Google 스프레드시트 생성 (예: '보드게임 결과')
2. 상단 메뉴: 확장 프로그램 → Apps Script
3. 이 폴더의 `apps-script.gs` 내용을 붙여넣기
4. 배포 → 새 배포 → 유형: 웹 앱
   - 실행 사용자: 나(교사)
   - 액세스 권한: "모든 사용자" (또는 "링크가 있는 사용자")
5. 배포 후 표시되는 웹 앱 URL을 teacher.html의 **결과 수집 URL**에 붙여넣기

### 2) 수업 운영
- 교사(teacher.html): 결과 수집 URL 입력 → 세션 코드 생성 → "학생 안내 복사"로 학생에게 전달
- 학생(student.html): 팀이름/세션코드/결과 전송 URL 입력 → 게임 진행
- 교사: "결과 새로고침"으로 세션 요약/오답을 확인, "AI 피드백 생성"으로 자동 피드백 생성

> 주의: Apps Script는 학교/계정 정책에 따라 웹 앱 배포 권한이 제한될 수 있습니다.
