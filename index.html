AI 주제 탐험 보드게임

사용방법(교사용)
1. 비밀번호 설정(교사용 화면에 처음 들어갈 때 사용할 비밀번호를 정해요)
2. 옵션 세팅(설정 버튼을 클릭하고 AI 모델, 문항 수, 활동 시간 등을 정해요)
3. 주제 입력(수업에 사용할 학습 주제를 입력해요)
4. 문제 생성(AI가 주제에 맞는 문제를 자동으로 만들어요)
추가 기능
교사용
  • 생성한 문제 파일 저장
  • 저장된 문제 파일 불러오기
학생용
  • 저장된 문제 파일 불러오기

어느 화면으로 들어갈까요?

교사용 (문제 만들기) 학생용 (게임 플레이)

학생용은 API 키가 필요 없습니다. 교사가 만든 문제가 자동 적용되어 바로 게임을 시작할 수 있고, 필요하면 “문제 파일(.json)”을 불러올 수도 있습니다.
